# ResumeMind – AI Resume Analyzer

ResumeMind is an AI-powered web application that analyzes resumes and provides structured feedback to help users improve their chances of passing Applicant Tracking Systems (ATS) and recruiter screening.
The platform evaluates uploaded resumes and generates insights such as ATS score, content feedback, and formatting suggestions. It also provides section-wise improvement tips to help users refine resume structure, readability, and keyword relevance.

# Application Workflow
User visits the homepage.

User uploads their resume through the upload interface.

The system processes the resume using AI analysis.

ResumeMind evaluates the document and generates insights.

Results page displays ATS score and improvement suggestions.
